Welcome to Gates Circuit!!

This app is used to visualize and construct circuits using logic gates.

This idea is inspired from a well talented YouTuber named Sebastian Lague. His videos are really helpful for understanding how computers work. Click on the links given below to watch some of his videos for better understanding on the usage of this app.

How do computers work?

How do computers remember?

Read the following instructions to know how to use this app:

> Nodes that are in the left side of the board are the input nodes.

> Nodes that are in the right side of the board are the output nodes.

- Add nodes on either side of the board.

- Draw wires on the board.

- Cut existing wires on the board.

- Create a GATE with the same functionality of the circuit made.

- Clear the board.

- Zoom in and out (Only during the absence of blocks and wires).

GATE NAME

- Gates available (Can be removed from the list by deleting it).

> Gates have nodes on 2 sides. The nodes on the left side are the input nodes for the gate and the nodes on the right side are the output nodes of the gate.

> Wires can be extended from the input nodes of the board or from the output nodes of the blocks. They can only be connected to the receiving nodes of the block and the output nodes of the board.

> State of the input nodes of the board can be changed by clicking on them.
